变量和赋值

2.1 变量和赋值概述

变量（Variable）

变量是用来存储数据的内存空间容器。可以把它想象成一个带有标签的盒子：

  • 有名字：通过变量名来访问这个盒子
  • 有类型：决定盒子里能放什么类型的数据（数字、文字等）
  • 存值：盒子里存放实际的数据

赋值（Assignment）

赋值是将具体的值存储到变量中的操作。使用等号 = 作为赋值运算符。

2.2 变量（整型）

2.2.1 变量声明（int类型）

变量声明是告诉编译器将要创建一个变量，并指定它的类型和名称。

内存分配过程：

  1. 编译时：编译器记录变量名和类型
  2. 运行时：JVM在栈内存中分配4个字节（32位）的空间
  3. 初始值：分配的空间中保留原有的随机值（可能是之前使用过的数据）

2.2.2 赋值语句

赋值语句是使用”=”操作符将一个值存储到变量中。

赋值过程：

  1. 计算右边的表达式（如果有）
  2. 将值转换为变量的类型（如果需要）
  3. 将值写入变量对应的内存位置

2.2.3 变量初始化

变量初始化是在声明变量的同时第一次给它赋值。

重要注意事项

  1. 局部变量必须初始化后才能使用，否则编译错误
  2. 成员变量有默认值（int默认为0）
  3. 每个赋值操作都是将新值写入变量的内存位置
  4. 变量可以多次赋值，但只能初始化一次
