Module1-JAVA简介

1.1 编程概述

1.1.1 计算机程序基础

计算机程序由一系列指令组成，这些指令依次执行，基本指令类型有：

  • 输入：程序获取数据，数据来源可以是文件、键盘、触摸屏、网络等。
  • 过程：程序对数据执行计算，例如将两个值相加，如 x + y。
  • 输出：程序将数据放入某个地方，例如文件、屏幕、网络等。

程序使用变量来引用数据，是因为当程序为变量（例如 x）赋新值时，变量的值也会随之改变。

1.1.2 算法

  • 算法是解决问题的方法和步骤，它关注的是逻辑。
  • 编程是把这些方法和步骤用计算机能理解的语言表达出来，它关注的是实现。

1.2 编程基础

1.2.1 第一个项目

  • 程序从 main() 方法开始，依次执行 main() 方法大括号 { } 内的语句。
  • 每个语句通常单独成行，以英文分号结尾。
  • 该int wage语句创建了一个名为 wage 的整型变量，wage = 20代表将 wage 赋值为 20。
  • print() 和 println() 方法会输出各种值。

1.2.2 基本输入

程序通常会获取输入值，对这些输入值进行处理，然后将输出值显示在屏幕或其他位置。输入值通常来自键盘、文件、网页表单或应用程序中的字段等。

Scanner是一种文本解析器，可以从键盘等输入源获取数字、单词或短语。

1.2.3 基本输出

输出文本

以下两个方法都是 Java 中用于控制台输出的基本方法，主要区别在于是否换行。

System.out.print() 连续输出在同一行。
System.out.println() 每个输出独占一行，可以单独使用输出空行。

输出文本的方式如下：System.out.print(“desired text”); 双引号内的文本称为字符串字面量。

输出变量值

使用同一条语句输出多个内容

程序员通常通过将文本、变量值和新行的输出合并，为每一行输出使用一条输出语句，只需用+符号分隔各项即可。

这种合并方式可以提升程序的可读性，因为程序代码与程序输出的对应关系更紧密。
